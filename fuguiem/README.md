# ふぐみかん IEM データベース 🎧

YouTubeチャンネル **[ふぐみかん](https://www.youtube.com/@fugumikan)**（中華ポータブルオーディオ・レビュー）で
紹介された **イヤホン（IEM）** を収集し、**特徴・価格（レビュー当時）・スペック** を一覧化。
**価格帯・ドライバ型式・音傾向・特徴タグ・ブランド・接続端子** で絞り込み検索できる
非公式のファンメイド・データベースサイト。

- フレームワーク: **Vue 3 + Vite**（外部API依存なしの静的SPA）
- データ: `src/data/iems.json` に同梱（51機種・20ブランド）
- 動画リンク: 各機種カードからチャンネル内の該当レビューへ遷移

## 使い方

```bash
npm install
npm run dev      # 開発サーバ (http://localhost:5173)
npm run build    # 本番ビルド → dist/（静的ホスティング可）
npm run preview  # ビルド結果をローカル確認
```

## 機能

| 絞り込み軸 | 内容 |
|-----------|------|
| フリーワード | 機種名・特徴・端子・筐体などを横断検索（スペース区切りでAND） |
| 価格帯 | 〜3千 / 3〜6千 / 6〜12千 / 12〜30千 / 30〜60千 / 60千〜（各件数表示） |
| ドライバ型式 | ダイナミック / マルチDD / ハイブリッド / トライブリッド / 平面駆動 / BA型 |
| 音傾向 | ハーマン・ニュートラル・ドンシャリ・ウォーム 等 |
| 特徴タグ | 高コスパ・名機・ボーカル・低音・モニター・リケーブル可・金属筐体 等 |
| ブランド | Moondrop / KZ / Truthear / 7Hz / SIMGOT ほか |
| 接続端子 | 0.78mm 2pin / 0.75mm 2pin(QDC) / MMCX / 直付け |
| 並び替え | 価格・新しさ・ドライバ数・名前 |

各機種カードには、ドライバ構成・インピーダンス・能率・再生周波数・端子・筐体・
音傾向・ふぐみかん流の短評・特徴タグ・レビュー動画リンクを掲載。

## データについて

- **価格** は各機の発売〜レビュー当時の国内実勢のおおよその値（税込・円）。為替・時期で変動するため目安。
- **能率(dB)** はメーカー表記準拠で、`/mW` と `/Vrms` の基準が混在する点に注意。
- 中身（スペック・価格・短評）は **人手でキュレーション** した `src/data/iems.json` を唯一の正とする。
- 本サイトは非公式であり、チャンネル運営者とは無関係です。

### データの追加・編集

`src/data/iems.json` の `iems[]` に1レコード追加するだけ。型式・価格帯・検索文字列などの
派生情報は表示時に `src/lib/iem.js` が自動計算する。追加後は検証を実行:

```bash
npm run validate   # 必須項目・id重複・ドライバ構成などをチェック
```

### 動画リンクの再生成（任意）

各カードの動画リンクは既定で「チャンネル内のその機種名での検索」に飛ぶ。
公開RSSフィードから実際のレビュー動画URL・投稿日を突き合わせて埋め込むこともできる:

```bash
npm run crawl            # ドライラン（マッチを表示）
node scripts/crawl.mjs --write   # iems.json に video.url / video.date を書き込む
```

> 実行環境から `www.youtube.com` へ到達できる必要があります（RSSは直近15本のみ）。
> ネットワーク制限のある環境では取得できず、その場合は検索リンクが使われます。

## 構成

| パス | 役割 |
|------|------|
| `src/data/iems.json` | 機種データ（唯一の正・人手キュレーション） |
| `src/lib/iem.js` | 派生情報の計算（型式分類・価格帯・検索インデックス等） |
| `src/components/IemCard.vue` | 機種カード |
| `src/App.vue` | 絞り込みUI＋一覧 |
| `scripts/crawl.mjs` | 動画リンク再生成（YouTube RSS突き合わせ） |
| `scripts/validate.mjs` | データ整合性チェック |
